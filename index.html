    Kinectable

    The kinectable gem implements a simple DSL for using a Kinect to recognize gestures.

    Installation

    Add this line to your application's Gemfile:

    gem 'kinectable'

    And then execute:

    $ bundle

    Or install it yourself as:

    $ gem install kinectable
